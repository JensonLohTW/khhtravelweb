故障排除步驟：

  1. 清除瀏覽器緩存：
    • Mac: 按 Cmd + Shift + R
    • Windows/Linux: 按 Ctrl + Shift + R
  2. 檢查開發者工具：
    • 按 F12 打開開發者工具
    • 查看 Console 標籤是否有 JavaScript 錯誤
    • 在 Elements 標籤中搜索 "qrToggle"
  3. 驗證 CSS 載入：
    • 在 Network 標籤中查看 style.css 是否成功載入（200 狀態碼）
    • 檢查 style.css 的大小是否為 ~19KB
